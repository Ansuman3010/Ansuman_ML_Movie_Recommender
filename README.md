# 🎬 Movie Recommender System

A content-based Movie Recommender System built using Python and
Streamlit that suggests similar movies based on user selection. The app
dynamically fetches movie posters using the TMDB API.

------------------------------------------------------------------------

## 🚀 Live Demo

https://ansumanmlmovierecommender-rsrfxzkgg4za5pcznfdoyt.streamlit.app/

------------------------------------------------------------------------

## 📌 Features

-   🎥 Select a movie from dropdown
-   🔍 Get Top 5 similar movie recommendations
-   ⭐ Movies sorted by weighted rating
-   🖼️ Dynamic movie posters using TMDB API

## 🧠 How It Works

1.  Uses a content-based filtering approach.
2.  Movies are vectorized based on metadata.
3.  Cosine similarity is computed between movies.
4.  Top similar movies are selected.
5.  Results are sorted by weighted rating.
6.  Posters are fetched dynamically from TMDB.
